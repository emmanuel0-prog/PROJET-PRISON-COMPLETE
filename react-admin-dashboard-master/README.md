# 🛡️ Système de Gestion Pénitentiaire Intelligent

### Par Emmanuel Mavungu

Bienvenue dans une plateforme moderne, sécurisée et performante de gestion pénitentiaire, développée par **Emmanuel Mavungu**, basée sur **React.js**, **Vite** et **Material UI**, intégrant des fonctionnalités avancées telles que l’identification biométrique par empreintes digitales.

Cette application vise à digitaliser, sécuriser et optimiser la gestion des établissements pénitentiaires, en offrant une interface intuitive et des outils puissants pour les administrations.

---

## 🚀 Fonctionnalités principales

### 📊 Tableau de bord (Dashboard)

Visualisation en temps réel des indicateurs clés :

* Nombre de détenus
* Taux d’occupation
* Activités récentes
* Alertes critiques

---

### 👥 Gestion du personnel

* Gestion complète des agents
* Suivi de présence
* Attribution des rôles et accès
* Profil détaillé du personnel

---

### 🔐 Authentification sécurisée

* Connexion avec identifiants sécurisés
* Authentification à double facteur (2FA)
* Gestion des sessions avec JWT
* Protection des routes (PrivateRoute)

---

### 🧬 Identification biométrique

* Enregistrement des empreintes digitales
* Identification rapide des détenus
* Sécurisation des accès sensibles
* Intégration avec les modules de contrôle

---

### 🧑‍⚖️ Gestion des détenus

* Enrôlement des détenus
* Dossiers complets (identité, judiciaire, médical)
* Affectation aux cellules
* Suivi des mouvements
* Gestion des libérations

---

### ⚖️ Gestion judiciaire

* Suivi des dossiers juridiques
* Alertes sur les échéances
* Historique des décisions judiciaires

---

### 🏥 Module médical

* Suivi médical des détenus
* Historique des consultations
* Gestion des traitements

---

### 🏢 Gestion carcérale

* Gestion des pavillons et cellules
* Contrôle des mouvements internes
* Cantine et gestion des ressources
* Surveillance et contrôle

---

### 👨‍👩‍👧‍👦 Gestion des visiteurs

* Enregistrement des visiteurs

* Système self-service

---

### 📅 Calendrier interactif

* Planification des activités
* Organisation des visites
* Gestion des événements internes

---

### 📈 Statistiques & visualisation

* Graphiques (Barres, Ligne, Camembert, Flux)

### 🌍 Module géographique

## ⚙️ Installation

1. Cloner le projet :

```bash
git clone https://github.com/ton-repo.git
```

2. Installer les dépendances :

```bash
npm install
```

3. Lancer le serveur :

```bash
npm run dev
```
